Enhanced text analysis tool that performs sentiment analysis, language detection, and content moderation with support for multiple languages and advanced features.

Features:
- Sentiment analysis with positive/negative/neutral classification
- Confidence scoring based on word coverage
- Language auto-detection (English, Spanish, French)
- Negation handling (e.g., "not good" becomes negative)
- Intensity modifier detection (e.g., "very good" boosts positive score)
- Subjectivity detection (subjective vs objective content)
- Expanded word dictionaries for better accuracy
- Multi-language support with localized word lists

The tool analyzes text for emotional tone, provides detailed breakdowns of detected words, and offers insights into the content's sentiment and language characteristics.
